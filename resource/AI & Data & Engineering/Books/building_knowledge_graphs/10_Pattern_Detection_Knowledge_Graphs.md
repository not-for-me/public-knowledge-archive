# 10. Pattern Detection Knowledge Graphs

## 챕터 개요 (3줄 요약)
- 지식그래프는 비즈니스 이벤트에서 비롯된 패턴을 발굴해 과거 통찰과 미래 예측 모두에 활용할 수 있는 강력한 플랫폼이다.
- 두 대표 유스케이스 — 사기 탐지(fraud detection)와 역량/팀 매칭(skills matching) — 를 통해 패턴매칭 + 그래프 데이터 과학의 결합을 보여준다.
- 핵심 패턴: 사기는 공유 식별자(전화·주소)로 연결된 합성 정체성 "fraud ring", 역량은 사람-프로젝트-스킬 그래프에서 시간·최신성으로 측정한 맥락적 전문성.

## Fraud Detection
> 사기는 막대하고($1당 추가 $4.23 손실), 표 형식 데이터는 구조를 가려 고객과 사기꾼 구분을 어렵게 한다 — 그래프가 연결 맥락을 드러낸다.

- 데이터 모델: CreditCard 중심, Purchase(NEXT로 시계열), Person-[:ACCOUNT_HOLDER], LIVES_AT(Address), OWNS(Phone). 정상 고객은 분리된 서브그래프.
- 검증: Person 수(2,399,999)와 커뮤니티 수를 비교 — Louvain 결과 커뮤니티가 약 50만 부족 → 일부 커뮤니티가 다인 = 의심.
- First-party fraud: 합법적으로 신용을 키운 뒤 한꺼번에 인출(cash-out). 수평 확장(가짜 정체성 다수)으로 신용한도 집계.

```
// fraud ring: 공유 전화/주소로 깊이 5+ 연결된 사람들
MATCH ring=(p1:Person)-[:OWNS|LIVES_AT*5..]-(p2:Person)
WHERE p1<>p2 RETURN ring
// 정상 가구(innocent bystander): 같은 주소+전화 공유
MATCH (p:Person)-[:LIVES_AT]->(:Address)<-[:LIVES_AT]-(:Person)-[:OWNS]->(:Phone)<-[:OWNS]-(p)
```

- Fraud ring vs 정상 가구 구분: 긴 체인(합성 정체성 망) vs 짧은 공유(가구). 무고한 가구를 사기로 낙인찍지 않도록 주의.
- 운영화: 거래 시점에는 저지연 서브그래프 로컬 패턴매칭, 고위험·고액은 더 넓은 영역의 알고리즘/데이터과학(수 ms~s)을 선별적 적용.
- 사례 Banking Circle: 커뮤니티 탐지로 ML 피처 생성(이웃 위험점수·조세회피처 거리 등)해 고위험 클러스터 탐지.

## Skills Matching
> 사람-프로젝트-스킬을 여러 계층(조직도·전문성·프로젝트이력·사회관계·부가스킬)으로 모델링해 질의 시점에 결합한다.

- 조직 지식그래프: org chart는 트리(REPORTS_TO) — 의도된 구조. 실제 협업 구조는 별개(비공식 조직도).
- 스킬 그래프: 단순 HAS_SKILL은 숙련도·최신성·실전 맥락이 없음 → PARTICIPATED_IN 프로젝트-REQUIRES 스킬로 누적 기간·최신성으로 진짜 역량 측정.
- 전문성 그래프: Slack/Teams 협업 데이터((User)-[:WROTE]->(Message)-[:MENTIONS]->(Skill))로 실질 전문가 발굴 — "knowledge power"(Andy Grove).
- 개인 성장: 함께 일한 동료가 익힌, 내가 안 익힌 스킬을 추천(커리어 경로). 사회 계층(RATED)으로 잘 맞는 팀 구성(cliques).
- 조직 성과 예측: FastRP 임베딩 + node regression(Random Forest)으로 미완료 프로젝트의 rating(성공도) 예측 → 조기 개입.
- 사례 DXC Technology(13만명): Career Navigator로 silo 연결, 유사 직원 스킬 갭 기반 커리어 추천.

> [모델링 관점 - 주식시장 도메인 적용]
> 이 장의 패턴 탐지 기법은 주식시장에 직접 대응된다: (1) Fraud ring 패턴 = 시세조종/차명거래 적발 — 공유 IP·계좌·디바이스·연락처로 연결된 "연계 거래자 망"을 깊이 N 체인으로 탐지하고, 한 계좌 적발 시 ring 전체를 즉시 추적해 동결. (2) Louvain/커뮤니티 탐지 = 비정상적으로 동조 매매하는 종목/계좌 클러스터(작전 세력) 식별. (3) 저지연 서브그래프 로컬 매칭 = 주문 체결 시점의 실시간 이상거래 스크리닝(거래 흐름에 ms 단위로 삽입), 고위험만 광역 알고리즘. (4) node regression 예측 = 과거 패턴(재무·관계 구조)으로 부실/디폴트 위험 종목 조기 경보. 핵심 교훈은 "표 데이터로는 안 보이는 연결 맥락"이 시장 감시·리스크의 차별적 신호이며, 인간 도메인 전문가가 패턴을 부트스트랩하고 그래프 데이터 과학이 이를 확장한다는 점이다.

## Summary (핵심 정리)
- 패턴 탐지 지식그래프는 사기 ring·역량 클러스터처럼 표 데이터로는 가려진 연결 패턴을 드러내며, 인간 전문성과 그래프 데이터 과학을 혼합해 풍부해진다.
- 운영에서는 저지연 서브그래프 로컬 매칭을 기본으로 하고 고위험에만 광역 알고리즘·ML을 선별 적용하는 것이 효율적이다.
- 주식시장에서는 연계 거래자 망 탐지·작전 세력 클러스터링·실시간 이상거래 스크리닝·부실 예측에 그대로 적용되며, 연결 맥락이 핵심 감시 신호다.
